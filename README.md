# Descodificador em JavaScript

## Este é um projeto de um Descodificador simples e bonito construído com JavaScript.

Como executar a aplicação
Para executar a aplicação, siga as etapas abaixo:

Autores
Victor - Trabalho inicial - https://github.com/VictorG-Reis

## Instalação

Abra um terminal e execute o comando:

```bash
git clone https://github.com/VictorG-Reis/Decodificador
```

Navegue até o diretório do projeto:

```bash
    cd Decodificador
```

Instale as dependências:

```bash
    npm install
```

Acesse a aplicação em seu navegador com server live ou o proprio index.html
